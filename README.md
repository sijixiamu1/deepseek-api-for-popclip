# deepseek-api-for-popclip

这是我的第一个程序... 🎉

## 📌 功能介绍

安装 [PopClip](https://www.popclip.app/) 之后，选中文本可以让 DeepSeek 进行解释 / 翻译等操作。

相比一般的 AI 插件，更加轻便快捷。

✨ 支持自定义 Prompt

---

## 🚀 使用方法

### 1️⃣ 安装 PopClip
👉 https://www.popclip.app/

### 2️⃣ 安装插件
双击打开 `DeepSeek.popcliptxt`，选择安装

### 3️⃣ 配置 API Key 和 Prompt
在 PopClip 中：

- 打开「我的插件」
- 点击右侧齿轮 ⚙️
- 输入你的 API Key：https://platform.deepseek.com/
- 设置 Prompt
